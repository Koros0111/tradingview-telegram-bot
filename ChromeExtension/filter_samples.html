Filter samples

There is several way to do filtering

Do not add a filter in the message

Example:

'Alert Message!'

If the "Ignore messages without filter specified" option is checked then this message will be ignored. This is a good way to filter out messages which you don't want to send into the Telegram (for example - AutoView messages).

If the "Ignore messages without filter specified" option is unchecked then this message will be sent using all keys.

* filter

Example:

[*]'Alert Message!'

This message will be sent to all keys.

Custom filter

Example:

[me]'Alert Message!'

This message will be sent to the key with id "me". You can specify several id's separated with comma:

[me,mychannel]'Alert Message!'
